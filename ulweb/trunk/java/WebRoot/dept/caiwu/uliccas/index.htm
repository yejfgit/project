新会计准则

　

新会计准则将于2007年1月1日起在合众人寿执行。这就意味着从2007年起，合众人寿将使用国际通用的商业语言与国际企业对话……在联合国国际会计标准专家工作组第22届会议上，财政部副部长王军称这套体系为“与中国国情相适应同时又充分与国际财务报告准则趋同的、涵盖各类企业各项经济业务、能够独立实施的会计准则体系”。
　
　 　 　 　
　 新会计准则的主要变化
 
变化一：会计基本原则； 变化二：会计要素的计量；
变化三：资产减值准备计提变革； 变化四：债务重组方法变革；
变化五：存货管理办法变革； 变化六：金融工具准则的变革；
变化七：合并报表基本理论的变革； 变化八：企业合并会计处理方法的变革

专家视点
　
如何推动会计准则体系顺利贯彻实施，任务更为艰巨。我们必须落实 “ 四个切实 ” 的工作计划，一是切实抓好宣传培训，二是切实做好实施准备，三是切实加强诚信建设，四是切实加强监督检查。
――财政部部长 金人庆
　
“会计准则将促使甚至迫使上市公司提高会计质量，讲清楚业务和结果是怎么样的，更好地保护投资者作为信息不够充分一方的利益。”
――财政部副部长会计准则委员会主席
楼继伟
　
　
中央企业要发挥带动和示范作用，要做执行新准则的表率，认真贯彻执行企业会计准则和审计准则体系，力争在2008年底之前全面执行。
――国务院国资委副主任 李伟
　
　
中国企业会计准则体系的发布是促进中国经济发展和提升中国的国际资本市场地位非常重要的一步。
――国际会计准则理事会主席
戴维・泰迪爵
　
合众人寿实施新会计准则体系对公司发展具有积极意义，一是有助于合众走国际化道理，二是有助于提高合众管理水平，三是有助于完善公司治理结构，四是有助于提升合众良好形象。
―― 合众人寿
